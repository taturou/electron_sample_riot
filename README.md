# これは何？

Electron + Riot.js (ES2015) のサンプルコードです。

# コンテンツ

Electron Quick Start と同じです。
node、chrome、electron のバージョンを表示します。

# インストール

```sh
$ git clone https://github.com/taturou/electron_sample_riot2.git
```

# ビルド

```sh
$ npm run build
```

# 実行

```sh
$ npm run serve
```

# 開発

以下を実行しておくことで、コードの修正を Livereload します。

```sh
$ npm run watch
```
